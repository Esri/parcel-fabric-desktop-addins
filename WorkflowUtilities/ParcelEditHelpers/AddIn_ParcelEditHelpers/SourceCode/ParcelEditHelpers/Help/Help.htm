Parcel Edit Helpers Add-in

The Parcel Edit Helpers Add-in provides sample code to third party developers for implementing custom functionality on parcel fabrics. The code can be found on GitHub. The add-in also provides non-developers some additional functionality for working with the Parcel Editor. This additional functionality is not supported by Esri support, however anyone using the samples or creating custom tools can add comments on this GitHub repository.

This help documentation provides information on the tools and features available with the add-in.

The functional features are found:

         directly on the add-ins toolbar

         on the parcel editor context menus, and appear as new commands

         on the editor sketch context menu, and appears as a new command

         as functionality that occurs based on other user actions (extensions that listen for events)

Toolbar

         Digitize Parcel use sketch tools such as trace and tangent curve to digitize a new parcel in the fabric

         Fabric Adjustment runs the fabric adjustment from a dock-able window.

         Options set options for the sample tools and other behavior.

         Help access to Help documentation.

Edit Sketch context menu

         Spiral right click in the map when using the edit sketch to access the Spiral command to enter the parameters for a spiral. (Start Radius, End Radius, Arc Length, Central Angle, etc.)

Parcel lines grid context menu

         Traverse gets closure information for selected lines. Lines in a construction are adjusted based on the currently set traverse adjustment method in the Parcel Editor options.

         Save Lines To File save the entries the lines grid to a comma delimited file

         Load Lines From File load lines into the lines grid from a traverse text file or a comma delimited text file.

         Pan To pan the map to the location of the line rows selected in the parcel lines and construction grids.

Extension

         Grid entry bearing recorder the values typed into the bearing field during construction and parcel entry are recorded in a field that is configured in the Parcel Edit Helper Add-in Options.
